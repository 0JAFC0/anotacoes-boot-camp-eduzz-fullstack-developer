o que são modulos? são arquivos js que tem a capacidade de exportar e importar informações de outros arquivos do mesmo tipo.

![](https://i.imgur.com/6GwESrM.png)

algumas vantagens são:

- organização do codigo;
- compartilhamento de variaveis em escopos diferentes;
- explicita as dependencias dos arquivos.
